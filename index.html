php-cloudDNS by snider

A PHP Binding to the Rackspace Cloud DNS API

This is a json centric PHP lib for the rackspace cloud dns service, which at the time of writing is in Beta so the lib will most likly break... however i will be using this myself so will update with bug fixes/ new features as and when i can... forks very much welcome

Dependencies

json, curl & PHP5

Install

just add the dns.php file to your project and create a new object from the class, auth will be done when the first call is made.
